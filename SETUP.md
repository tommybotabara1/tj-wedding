# New Machine Setup Guide

This project uses the **WAT framework** (Workflows, Agents, Tools) with Claude Code as the AI agent. Follow these steps to get it running on a fresh machine.

---

## 1. Clone the repo

```bash
git clone https://github.com/tommybotabara1/tj-wedding.git
cd tj-wedding
```

---

## 2. Install dependencies

**Python 3.11+** is required. Install packages used by the tools:

```bash
pip install gspread google-auth google-auth-oauthlib openpyxl
```

---

## 3. Restore secrets (NOT in the repo — keep these safe)

Two files are gitignored and must be restored manually:

### `.env`
Create a `.env` file in the project root:
```
GOOGLE_DRIVE_FILE_ID=<your Google Sheet file ID>
```
The file ID is the long string in the Google Sheets URL.

### `credentials.json`
This is the Google OAuth client credentials file:
1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Open the project used for this repo (or create one)
3. Enable **Google Sheets API** and **Google Drive API**
4. Create an **OAuth 2.0 Client ID** (Desktop app type)
5. Download the JSON and save it as `credentials.json` in the project root

On first run of any tool that calls Google APIs, it will open a browser for OAuth. After you approve, `token.json` will be created automatically.

---

## 4. Install Claude Code

```bash
npm install -g @anthropic/claude-code
```

Then authenticate:
```bash
claude
```

---

## 5. Open the project in Claude Code

```bash
cd tj-wedding
claude
```

Claude will read `CLAUDE.md` automatically and understand the project structure and WAT framework conventions.

## Project Structure

```
docs/           # Wedding website (HTML — deployed via GitHub Pages)
planning/       # Guest list Excel, floor plan PDF, seating mockup, pegs
tools/          # Python scripts for deterministic execution
workflows/      # Markdown SOPs — read these before running tools
.env            # API keys (gitignored — restore manually)
credentials.json  # Google OAuth (gitignored — restore manually)
CLAUDE.md       # Instructions for Claude Code agent
README.md       # Project overview and reference docs
```

## Notes

- `.claude/` is gitignored — Claude Code will recreate its local settings on first run
- `.tmp/` is gitignored — all temp files are regenerated by tools
- Always run `apply_guest_colors.py` after manual edits to the Excel guest list
- Never run `build_guest_list.py` unless rebuilding from scratch from the source file
